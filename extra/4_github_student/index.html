GitHub Student Developer Pack

Cómo inscribirse en el programa para estudiantes de GitHub

Overview

En esta sección explicaremos el proceso para inscribirse al programa GitHub Student Developer Pack, que ofrece a los estudiantes acceso gratuito a varias herramientas y servicios premium que normalmente tienen un coste.

Tutorial: Inscripción en GitHub Student Developer Pack

Sigue estos pasos para obtener los beneficios de estudiante en GitHub:

  1. Accede a la Página: Abre tu navegador web y ve a la página oficial de GitHub Education para estudiantes: https://github.com/education/students
  2. Inicia el Proceso: Busca y haz clic en el botón principal que suele decir "Sign up for Student Developer Pack" o "Get benefits".
  3. Inicia Sesión o Crea una Cuenta:
    • Si ya tienes una cuenta de GitHub, inicia sesión con tus credenciales.
    • Si no tienes cuenta, haz clic en "Create an account" y sigue los pasos para registrarte antes de continuar con la solicitud de estudiante.
  4. Verifica tu Estado de Estudiante: Se te redirigirá a un formulario donde deberás proporcionar información para verificar que eres estudiante. Prepara lo siguiente:
    • Correo Electrónico Académico: Puedes usar tu correo personal, pero sería más interesante añadir el correo de la universidad como correo alternativo en GitHub: Añadir correo a cuenta GitHub
    • Nombre del Centro Educativo: Escribe el nombre completo de la escuela, en este caso OBS Business School
    • Prueba de Matriculación: Deberás subir un documento que demuestre que eres estudiante actualmente. Asegúrate de que incluya una fecha válida (del año académico en curso). Puede valer con la prueba del pago de la matrícula.
    • Plan de Uso: Describe brevemente cómo planeas usar GitHub, aquí habla que usaremos GitHub en varias de las asignaturas del Máster.
  5. Envía tu Solicitud: Revisa que toda la información y los documentos subidos sean correctos. Haz clic en el botón para enviar tu solicitud (puede decir "Submit", "Process my application" o similar).
  6. Espera la Confirmación: GitHub revisará tu solicitud. Este proceso puede tardar desde unos pocos días hasta algunas semanas, dependiendo del volumen y la claridad de tu prueba. Recibirás un correo electrónico notificándote si tu solicitud ha sido aprobada.

Beneficios del GitHub Student Developer Pack

Una vez aprobada tu solicitud, tendrás acceso a numerosos beneficios, entre ellos:

  • GitHub Pro de forma gratuita mientras seas estudiante
  • Créditos en servicios en la nube como AWS, Azure, y Digital Ocean
  • Acceso a herramientas de desarrollo como JetBrains, GitKraken, y más
  • Acceso a plataformas de aprendizaje como Frontend Masters o DataCamp

Estos beneficios te serán de gran utilidad durante tu formación, permitiéndote acceder a herramientas profesionales sin coste adicional.
